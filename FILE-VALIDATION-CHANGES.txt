## Key Features Added

1. File Type Validation
   - MIME type whitelist enforcement
   - Extension validation (case-insensitive)
   - Audio/video specific validation

2. File Size Validation
   - 100MB limit for audio/video files
   - 10MB limit for general files
   - Configurable custom limits

3. Security Improvements
   - Prevents malware uploads (exe, sh, js blocked)
   - Prevents storage exhaustion attacks
   - Multiple validation layers (defense in depth)
   - Clear, actionable error messages

4. Developer Experience
   - Easy-to-use validator functions
   - Comprehensive test coverage
   - Detailed documentation
   - Utility functions (formatBytes, getFileExtension)

## Integration Points

1. Voice Notes Upload (API Gateway)
   - POST /voice-notes/upload
   - Validates file before processing
   - Rejects invalid types/sizes immediately

2. Presigned URL Generation (Files Service)
   - GET /files/upload-url/:key
   - Validates file extension
   - Returns maxSizeBytes for client validation

3. Common Library
   - Exports all validators via @clinic/common
   - Available to all services
   - Type-safe with TypeScript
