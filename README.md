# Clinic App

## Auth Service

### Setup

```bash
cp .env.example .env
cd services/auth-service
npm install
```

### Run

```bash
npm run start:dev
```

### Test

```bash
npm test
```

## Frontend

### Setup

```bash
cp frontend/.env.example frontend/.env
cd frontend
npm install
```

### Run

```bash
npm run dev
```

### Test

```bash
npm test
```
